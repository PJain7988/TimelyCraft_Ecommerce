# TimelyCraft ⌚

TimelyCraft is a premium e-commerce website for luxury and smart watches. This project has been redesigned to offer a professional, dark luxury user experience with responsive design, smooth animations, and enhanced interactivity.

## ✨ Features

- **Premium Dark Luxury Theme:** A cohesive design system utilizing dark backgrounds (`#0a0a0f`) and elegant gold (`#c9a84c`) accents.
- **Glassmorphism UI:** Modern frosted-glass header and overlay effects using `backdrop-filter: blur(20px)`.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices with a custom hamburger menu.
- **Smooth Animations:** Scroll-reveal animations, animated counters, hover lifts, and glowing effects.
- **Dynamic Product Sections:** Product carousels with arrow navigation and auto-play, plus tabbed product details (description, specifications, reviews).
- **Functional Checkout Flow:** A complete multi-step checkout process (Cart -> Details -> Payment -> Confirmation) with form validation and local storage integration for cart data.
- **SEO Optimized:** Meta descriptions and Open Graph tags included across all pages.

## 🛠️ Technologies Used

- **HTML5:** Semantic and accessible page structures across 11 unique templates.
- **CSS3:** Vanilla CSS with custom properties (variables) for a highly maintainable token-based design system.
- **JavaScript (Vanilla):** Enhanced DOM manipulations, IntersectionObserver for scroll animations, dynamic checkout logic, and localStorage state management.
- **Typography:** Playfair Display for headings and Inter for body text (via Google Fonts).
- **Icons:** Font Awesome 6.5 for scalable, high-quality vector icons.

## 📁 Project Structure

- `index.html` - Homepage featuring hero, categories, stats, and testimonials.
- `shop.html` - All products listing.
- `collections.html` - Curated watch collections.
- `product.html` - Detailed single product view.
- `blog.html` - Latest articles and watch news.
- `blog-single.html` - Individual article reading view.
- `cart.html` - Shopping cart interface.
- `checkout.html` - Multi-step secure checkout process.
- `contact.html` - Contact form and company information.
- `about.html` - Company history and timeline.
- `wishlist.html` - Saved items for later purchase.
- `CSS/style.css` - The master stylesheet for the entire application.
- `JS/script.js` - Global interactive logic and component behaviors.

## 🚀 How to Run

1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Simply open `index.html` in your preferred web browser (e.g., Chrome, Edge, Safari). No build tools or local servers are strictly required, though a local server (like VS Code Live Server) is recommended for the best experience.
